ROS2执行服务测试命令
===================

在Ubuntu电脑上运行以下命令测试服务：

1. 启动服务
----------
cd /path/to/ros2_server
npm install
npm start

2. 健康检查
----------
curl http://localhost:3001/health

期望返回：{"status":"ok","message":"ROS2执行服务运行正常"}

3. 测试ROS2命令执行（移动机器人）
-----------------------------
curl -X POST http://localhost:3001/execute \
  -H "Content-Type: application/json" \
  -d '{"command":"ros2 topic pub /cmd_vel geometry_msgs/msg/Twist \"{linear: {x: 1.0, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 0.5}}\" --once"}'

期望返回：{"success":true,"output":"...","warning":"..."}

注意事项：
- 服务监听端口3001
- 只允许执行以 "ros2 topic pub" 开头的命令
- 命令执行超时时间为10秒
- 需要在有ROS2环境的Ubuntu系统上测试
